YT Chat Translator Privacy Policy

Last updated: May 21, 2026

YT Chat Translator translates YouTube comments and live chat inside the user’s browser.

Information We Use

The extension may store settings such as selected language, translation engine, display mode, color preferences, usage counters, sign-in token, and account email locally in Chrome storage.

How Data Is Processed

Text selected from YouTube comments or live chat is sent only to the translation provider chosen in the extension, such as Google Translate, Microsoft Translator, or Yandex Translate, to return a translation.

Google Sign-In

If the user signs in, the extension uses Chrome identity only to authenticate the user and confirm account status. It does not request access to Gmail, Drive, YouTube account management, or other private Google data.

Data Sharing

We do not sell personal data. We do not use user data for advertising. Translation text may be processed by the selected translation provider according to that provider’s terms and privacy policy.

Remote Code

The extension does not execute remote code. Network access is used for YouTube pages, translation requests, and account verification features.

Contact

For privacy questions, contact the developer through the Chrome Web Store listing or the project repository.
